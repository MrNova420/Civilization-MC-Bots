🔌 Server Connection

Configure which Minecraft server the bot should connect to. Changes will reconnect the bot.

Enter server IP or domain (e.g., play.hypixel.net, mc.server.com, 192.168.1.100)
Default Minecraft port is 25565
Leave empty for auto-detection (recommended)
💡 Quick Tips:
• For offline/cracked servers: Just enter the IP
• For online servers: Make sure your account is configured in auth settings
• Public servers: play.hypixel.net, mc.hypixel.net, 2b2t.org
• Local server: localhost or 127.0.0.1
